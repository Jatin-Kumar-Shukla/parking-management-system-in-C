Code

3. Install GCC:

You can install GCC using either of the following commands:
Install GCC directly:
Code
sudo apt install gcc
This command installs the GCC compiler itself and Install build-essential package (recommended).
code
sudo apt install build-essential

4. Verify Installation:

After the installation is complete, verify that GCC is installed correctly by checking the compiler version:

sudo apt install g++

The other software shows a menu with many options to enter for different vehicles, which is not convenient for the user. At the time of vehicle arrival and at the time of departure, the time was entered manually in the system so it could be wrongly entered by the admin because human mistakes are very common.this system print the time of the system on screen whenever required not need to entered by the user manually
